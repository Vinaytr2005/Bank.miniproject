🏦 Python Banking System (CLI Project)

A simple command-line based banking application built using Python that allows users to perform basic banking operations such as depositing money, withdrawing money, and checking account balance.

🚀 Features
💰 Deposit money into account
💸 Withdraw (debit) money from account
📊 Check current account balance
🔒 Basic validation (e.g., insufficient balance handling)
🧾 Simple and interactive CLI interface
🛠️ Technologies Used
Python 3
Core Python concepts (functions, conditionals, loops)
📌 How It Works

The program runs in a loop and displays a menu like:

1 → Deposit Money  
2 → Withdraw Money  
3 → Check Balance  
4 → Exit  

Users can choose an option and perform transactions interactively.

▶️ Usage
1. Clone the repository
git clone https://github.com/Vinaytr2005/Bank.miniproject.git
cd Bank.miniproject
2. Run the program
python your_script_name.py
💡 Example Workflow
Select Deposit → Enter amount → Balance updated
Select Withdraw → Enter amount → Deducted if sufficient balance
Select Check Balance → Displays current balance
⚠️ Validations Implemented
Prevents withdrawing more than available balance
Handles invalid inputs
Ensures smooth user interaction
